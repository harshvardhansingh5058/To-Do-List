# To Do List App

A clean and responsive To Do List application built using HTML, CSS, and JavaScript.
This project helps users manage daily tasks with features like task creation, editing, deleting, task completion tracking, priority badges, dark mode, and live task statistics.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Priority system (Low, Medium, High)
* Dark mode UI
* Responsive design
* Live task statistics
* Local storage support

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage 
* Font Awesome Icons
